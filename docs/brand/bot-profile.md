# Профиль бота LOOP

## Имя

LOOP

## Короткое описание

Живой цикл: очередь BANK, равные DUEL один на один и рейтинг подтверждённых действий.

## Полное описание

LOOP — живой цикл внутри Telegram. В BANK новые взносы постепенно наполняют более ранние позиции прозрачной очереди. В DUEL два участника принимают равный вызов один на один. Рейтинг показывает завершённые циклы и надёжность. Все действия подтверждаются внешним TON-кошельком.

## Сообщение `/start`

Бот последовательно показывает одному пользователю все 32 коротких варианта:
новый текст при каждом `/start`, без повторов до завершения полного круга.
Счётчик хранится между перезапусками приложения. Базовый вариант:

∞ LOOP

Цикл уже идёт.

BANK — войди в очередь.

DUEL — брось вызов.

Твой ход.

Кнопка: `ОТКРЫТЬ LOOP`

## Команда `/support`

Показывает краткий порядок действий при ошибке, предупреждает не раскрывать
seed-фразу, приватный ключ, пароль или код из Telegram и открывает настроенный
`LOOP_SUPPORT_URL`.
