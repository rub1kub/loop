# Профиль бота LOOP

## Имя

LOOP

## Короткое описание

Живой цикл: очередь BANK, равные DUEL один на один и рейтинг подтверждённых действий.

## Полное описание

LOOP — живой цикл внутри Telegram. В BANK новые взносы постепенно наполняют более ранние позиции прозрачной очереди. В DUEL два участника принимают равный вызов один на один. Рейтинг показывает завершённые циклы и надёжность. Все действия подтверждаются внешним TON-кошельком.

## Сообщение `/start`

∞ LOOP

Ты вошёл в живой цикл.

BANK — прозрачная очередь, в которой каждый новый взнос продолжает движение и постепенно наполняет более ранние позиции. Ты выбираешь сумму и цель, а состояние цикла всегда видно в приложении.

DUEL — равный вызов один на один. Одинаковые ставки, понятные условия и результат, который нельзя изменить задним числом.

РЕЙТИНГ — твой след в LOOP: завершённые циклы, дуэли и надёжность. Размер ставки не делает участника выше остальных.

Кошелёк остаётся внешним. LOOP использует его только для подтверждения действий и получения выплат.

Цикл уже идёт. Твой ход — войти.

Нужна помощь — `/support`

Кнопка: `ОТКРЫТЬ LOOP`

## Команда `/support`

Показывает краткий порядок действий при ошибке, предупреждает не раскрывать
seed-фразу, приватный ключ, пароль или код из Telegram и открывает настроенный
`LOOP_SUPPORT_URL`.
